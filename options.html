SEO Brein Translator

Automatically translate community.seobrein.nl content to English

How it works

This extension automatically translates Dutch content to English on community.seobrein.nl using an unofficial Google Translate service.

Translation Settings

Chrome Auto-Translate Compatibility

This extension works great with Chrome's built-in auto-translate feature. Chrome will handle most of the page translation, and this extension will specifically target elements that might be missed by Chrome's auto-translate.

Add CSS selectors for elements you want to translate. Each selector should be on a new line.

Usage Instructions

  1. Configure the CSS selectors above to target the elements you want to translate
  2. Navigate to any page on community.seobrein.nl
  3. Enable Chrome's built-in auto-translate feature if desired (Chrome will offer to translate the page)
  4. This extension will automatically detect and translate elements matching your selectors
  5. The extension works as a complement to Chrome's auto-translate, focusing on specific elements that might be missed
