SEO Brein Translator

Automatically translate community.seobrein.nl content to English

How it works

This extension automatically translates Dutch content to English on community.seobrein.nl using Chrome's built-in Translation API. If that's not available, it falls back to Google Cloud Translate (requires API key below).

Translation Settings

Only needed if Chrome's built-in translator is unavailable. Get your API key from the Google Cloud Console.

Usage Instructions

  1. Navigate to any page on community.seobrein.nl
  2. The extension will automatically detect and translate Dutch content to English
  3. Translation happens in real-time, including dynamically loaded content
  4. No user interaction required - it works automatically!
